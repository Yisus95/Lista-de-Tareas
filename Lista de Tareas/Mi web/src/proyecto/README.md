## Cómo ejecutar (local)
1. Clona:
   git clone https://github.com/Yisus95/Lista-de-Tareas.git
2. Entra al proyecto:
   cd Lista-de-Tareas
3. Crea la rama de trabajo:
   git checkout -b chore/add-project-config

(Instalar Node y dependencias es opcional; recomendado)
4. Instalar dependencias:
   npm install

Comandos útiles:
- `npm run lint` — Ejecuta ESLint
- `npm run format` — Ejecuta Prettier
- `npm test` — Ejecuta tests (Jest)
